# SnakeGame

#Snake Game in Java

Simple Snake Game built using Java Swing.

## Features
- Arrow key movement
- Score tracking
- Game Over screen
- Snake grows after eating food

##Run
Option 1 :
javac src/SnakeGame.java
java -cp src SnakeGame

Option 2:
javac -encoding UTF-8 SnakeGame.java
java SnakeGame


## Tech
Java, Swing, OOP
